## help

- help 명령어를 단독으로 입력하면 cmd 가 제공하는 모든 명령어를 보여줍니다.
  
```python
$ help  
``` 
- help 의 매개변수로 명령어를 지정하면 그것에 대한 도움말을 보여줍니다.

```python
$ help <명령어>   
```

- 명령어의 도움말은 help 명령을 사용하는 대신 /? 를 지정해서 볼 수도 있습니다.
  
```python
$ <명령어> /?  
```
